# tms

`tms` easily selects and creates tmux sessions using fzf

## Usage

### Command

```
$ tms
```

### Widget

Allocate to shortcut key

```.zshrc
bindkey '^[t' tms
```

## Installastion

#### antigen users

```
antigen bundle yuki-ycino/tms
```

#### zgen users

```
zgen load yuki-ycino/tms
```
