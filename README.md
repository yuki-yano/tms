# tms
`tms` easily selects and creates tmux sessions using fzf

## Usage

### Command

```
$ tms
```

### Widget

Allocate to shortcut key

```.zshrc
bindkey '^[t' tms
```

## Installastion

#### zplug users

```zsh:.zshrc
zplug "yuki-ycino/tms", as:command
```

#### antigen users

```
antigen bundle yuki-ycino/tms
```

#### zgen users

```
zgen load yuki-ycino/tms
```

## Requirements
Either of these is necessary.

+ `fzf-tmux`
